# GradientBG
Gradient Background Generator - Code and Visualization.

You can test different kind of gradients on this site.
You get the code for them in real time.

## I. Main Features
**1.** Direction selection  
**2.** Color quantity selection  
**3.** Color selection  
**4.** Section selection
